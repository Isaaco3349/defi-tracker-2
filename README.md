# DeFi Tracker 2

A DeFi portfolio tracker built with Next.js and TypeScript that lets you monitor your wallet balances across multiple chains.

## Tech Stack
- Next.js 14
- TypeScript
- CSS Modules

## Live Demo
[defi-tracker-2.vercel.app](https://defi-tracker-2.vercel.app)

## Getting Started
```bash
npm install
npm run dev
```
